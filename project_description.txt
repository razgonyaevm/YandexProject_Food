Описание проекта Яндекс Лицей

При запуске появляется приветственное окно с вариантами выбора действия:
1. Изменение элементов базы данных
2. Удаление элементов базы данных
3. Добавление элементов базы данных
4. Кнопка "Сделать заказ"
5. Кнопка выхода из приложения

При нажатии любой кнопки будет закрываться данная приветственная форма и открываться новая.
